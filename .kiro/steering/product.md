### Product overview – Python tools

This workspace produces Python-based command-line tools.

All applications should:

- Be exposed as CLIs with a clean, discoverable interface.
- Provide useful, well-formatted console output by default.
- Be easy to install and run using modern Python tooling.
- Be simple to test, lint, and format in a consistent way across projects.

Wherever possible, prefer standard, well-maintained libraries and patterns, and keep the developer ergonomics of the tools front and center.
